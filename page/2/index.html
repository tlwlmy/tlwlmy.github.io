VirtualBox Usb设备

发表于 2016-01-05   |  

Ubuntu 14.04 VirtualBox虚拟机win7支持Usb设备

需要从官网下载Oracle VM VirtualBox Extension Pac
扩展插件需要安装对应的版本，不然可能会报错,查看VirtualBox 版本

下载完成后点击安装，在USB设置添加USB筛选器
查询自己当前用户

将当前用户加入vboxusers组

重启Ubuntu系统，再启动虚拟机win7
